# 基于正交设计、均匀设计与响应面法的激光焊接工艺参数多方法优化与比较研究

## 摘要

本文针对激光焊接工艺中多因素多水平的参数优化问题，分别采用正交试验设计、均匀设计和响应面法三种经典的试验设计方法，以激光功率、焊接速度、离焦量和保护气流量为试验因素，以焊接接头抗拉强度为响应指标，系统开展了工艺参数的优化研究。正交设计选用L9(3^4)正交表，通过极差分析和方差分析确定了因素的主次关系与最优水平组合；均匀设计采用U9*(9^4)均匀设计表，通过多元线性回归建立了预测模型；响应面法采用Box-Behnken设计，构建了包含线性项、二次项和交互项的二阶回归模型，并通过网格搜索法确定了全局最优参数。最后对三种方法的试验效率、模型精度和优化结果进行了综合比较。结果表明：激光功率是影响抗拉强度的最显著因素，三种方法得到的最优参数组合高度一致，综合推荐最优工艺参数为激光功率1400W、焊接速度40mm/s、离焦量+1mm、保护气流量20L/min。

**关键词**：正交试验设计；均匀设计；响应面法；激光焊接；参数优化；方差分析

## 1 引言

### 1.1 研究背景与意义

激光焊接作为一种高精度、高效率的先进连接技术，已广泛应用于航空航天、汽车制造、电子封装等领域。焊接质量直接取决于工艺参数的合理选择，而激光焊接过程涉及激光功率、焊接速度、离焦量、保护气流量等多个参数，且各参数之间存在复杂的交互作用。

传统的"单因素轮换法"通过固定其他因素、逐一改变单因素来寻找最优条件，这种方法不仅试验次数多、周期长，而且无法揭示因素间的交互作用。试验设计（Design of Experiments，DOE）方法以数理统计为理论基础，能够以最少的试验次数获取最大的信息量，已成为现代工业优化的重要手段。

### 1.2 试验设计方法概述

根据教材《试验设计方法》（赵选民，科学出版社），试验设计的核心方法体系包括：

- **方差分析（ANOVA）**：用于判断各因素对试验指标影响的显著性；
- **回归分析**：用于建立试验指标与影响因素之间的定量关系模型；
- **正交试验设计**：利用正交表安排多因素多水平试验，具有"均匀分散、齐整可比"的特点；
- **均匀设计**：基于数论方法，使试验点在试验空间内均匀分布，适用于多水平大范围的试验；
- **响应面法**：通过建立二阶回归模型，拟合响应曲面，精确寻找最优参数。

本文综合运用上述方法，对一个典型的激光焊接工艺参数优化问题进行系统研究，并比较三种设计方法的优劣与适用条件。

### 1.3 问题描述

某激光焊接工艺需优化四个可调参数以求最大化焊接接头抗拉强度（MPa）：

| 因素 | 符号 | 单位 | 水平1 | 水平2 | 水平3 | 水平范围 |
|------|------|------|-------|-------|-------|----------|
| 激光功率 | A | W | 1000 | 1200 | 1400 | 1000-1400 |
| 焊接速度 | B | mm/s | 20 | 30 | 40 | 20-40 |
| 离焦量 | C | mm | -1 | 0 | +1 | -1~+1 |
| 保护气流量 | D | L/min | 10 | 15 | 20 | 10-20 |

其中均匀设计将每个因素扩展到9个水平，响应面法使用3个编码水平(-1, 0, +1)。

### 2.4 方差分析

#### 2.4.1 总偏差平方和的分解

试验总次数n=9，总平均值ȳ=(350.0+383.0+...+456.0)/9=414.3 MPa。

- 总偏差平方和：SS_T = Σ(yᵢ-ȳ)² = 8952.0
- 修正项：CT = T²/n = 414.3²×9 = 1545232.1（用于计算各因素SS）

各因素的偏差平方和：

SS_A = (K₁²+K₂²+K₃²)/3 - CT = (1140²+1245²+1344²)/3 - CT = 6938.0

SS_B = (1197²+1251²+1281²)/3 - CT = 1208.0

SS_C = (1211²+1241²+1277²)/3 - CT = 728.0

SS_D = (1231²+1246²+1252²)/3 - CT = 78.0

验证：SS_A+SS_B+SS_C+SS_D = 6938+1208+728+78 = 8952 = SS_T（饱和设计，无纯误差）。

#### 2.4.2 显著性检验

由于L9(3^4)为饱和设计（总自由度8，4个因素各2个自由度），无剩余自由度用于误差估计。将偏差平方和最小的因素C和D合并作为误差项：

SS_e = SS_C + SS_D = 728.0 + 78.0 = 806.0
df_e = df_C + df_D = 2 + 2 = 4
MS_e = SS_e/df_e = 806.0/4 = 201.5

最优方案为A₃B₃C₃D₃。对显著因素A取其最优水平A₃，B、C、D取各自均值最大的水平B₃、C₃、D₃。

工程平均（最优条件下抗拉强度的点估计）：

μ_opt = ȳ + (k₃A-ȳ) + (k₃B-ȳ) + (k₃C-ȳ) + (k₃D-ȳ)
      = 414.3 + (448.0-414.3) + (427.0-414.3) + (425.7-414.3) + (417.3-414.3)

### 3.3 多元线性回归建模

对均匀设计数据建立多元一次回归模型：

Y = β₀ + β₁·A + β₂·B + β₃·C + β₄·D + ε

采用最小二乘法拟合得到回归方程：

**Y = 0.06 + 0.2257A + 0.2540B + 0.1404C + 2.4815D**

模型判定系数**R² = 0.9951**，表明回归方程对试验数据的拟合度极高，自变量解释了99.51%的响应变异。

回归系数分析：
- β₁=0.2257：激光功率每增加1W，抗拉强度增加约0.23 MPa（在1000-1400W范围），弹性最大；
- β₂=0.2540：焊接速度每增加1mm/s，强度增加约0.25 MPa，但因速度量纲小、变化范围窄，实际综合效应小于功率；
- β₃=0.1404：离焦量影响相对较小；
- β₄=2.4815：气体流量的系数绝对值较大，但其整体变化幅度有限（10-20L/min）。

### 3.4 最优参数预测

根据回归方程，所有系数均为正值，因此各因素均取最大值时可获得最高抗拉强度。代入最优参数A=1400W、B=30mm/s（取适中值）、C=1mm、D=20L/min：

Y_pred = 0.06 + 0.2257×1400 + 0.2540×30 + 0.1404×1 + 2.4815×20 = 373.4 MPa

## 4 响应面法设计与分析

### 4.1 Box-Behnken设计

对于4因素（k=4）的BBD设计，总试验次数为2k(k-1)+n_c = 2×4×3+3 = 27次，其中24个阶乘点加3个中心点（用于估计纯误差）。

### 4.2 二阶响应面模型拟合

建立包含线性项、二次项和交互项的二阶多项式回归模型（编码空间）：

Y = β₀ + Σβᵢxᵢ + Σβᵢᵢxᵢ² + Σᵢ<ⱼβᵢⱼxᵢxⱼ + ε

采用最小二乘法拟合27次试验数据，得到回归方程：

**Y = 430.7 + 34.5x₁ + 13.6x₂ + 10.4x₃ + 4.6x₄
    - 5.8x₁² - 1.8x₂² - 1.2x₃² + 1.0x₄²
    + 5.2x₁x₂ + 0.3x₁x₃ - 1.1x₁x₄ - 0.7x₂x₃ - 1.2x₂x₄ - 0.6x₃x₄**

模型判定系数**R² = 0.9931**，拟合效果优良。

### 4.3 模型分析

**线性项分析：**
系数符号均为正，表明四个因素在试验范围内对强度均为正向影响，其中x₁（激光功率）的系数34.5最大，是第二因素x₂（焊接速度）的2.5倍。

**二次项分析：**
x₁²的系数为-5.8（负值），说明激光功率对抗拉强度的影响存在边际效应递减——随着功率继续增大，强度提升速率放缓。B、C同样呈现该趋势，但幅度较小。

**交互项分析：**
- x₁x₂（功率×速度）交互项系数为+5.2，为最大的交互项，表明激光功率和焊接速度之间存在协同效应；
- 其他交互项系数接近0，交互作用不明显。

**响应面图分析（略）：** 以x₁和x₂为坐标轴绘制3D响应曲面和等高线图，呈现出鞍形曲面特征，曲面的脊线方向指向高功率-高速度区域，在该区域抗拉强度达到最大值。

### 4.4 最优参数求解

采用网格搜索法（步长0.05，xᵢ∈[-1,+1]），在编码空间内搜索使Y最大化的点。

最优编码解：x₁=1.000, x₂=1.000, x₃=1.000, x₄=1.000

解码后最优参数：
- 激光功率 A = 1400 W
- 焊接速度 B = 40 mm/s
- 离焦量 C = +1.00 mm
- 保护气流量 D = 20 L/min

预测最优抗拉强度：**Y_max = 487.7 MPa**

### 5.2 综合评分法

对两个指标进行归一化处理，采用加权综合评分法：
- 抗拉强度Y₁（望大）：Y₁' = (Y₁-Y₁_min)/(Y₁_max-Y₁_min)
- 焊缝宽度偏差（望小）：Y₂_dev = |Y₂-1.5|， Y₂' = 1 - Y₂_dev/Y₂_dev_max
- 综合评分 Z = 0.6×Y₁' + 0.4×Y₂'（权重视工艺需求而定）

综合评分极差分析表明：最优方案仍为A₃B₃C₃D₃，与单指标分析结论一致。

### 6.1 比较分析

**表6 三种试验设计方法综合比较**

| 比较维度 | 正交试验设计L9(3^4) | 均匀设计U9*(9^4) | 响应面法BBD |
|----------|-------------------|-----------------|------------|

### 6.2 各方法特点总结

**(1) 正交试验设计的优势：**
- 试验次数少（9次即可处理4因素3水平）
- 计算简单，极差分析和方差分析均可手工完成
- 具有"整齐可比"性，便于直观判断最优水平
- 局限性：无法拟合连续的响应面，难以找到非节点上的最优解

**(2) 均匀设计的优势：**
- 同样试验次数下可考察更多水平（9水平 vs 3水平）
- 试验点在空间内分布最均匀
- 适合大范围、多水平的探索性试验
- 局限性：需依赖回归分析，结果不够直观；不宜考察交互作用和二次效应

**(3) 响应面法的优势：**
- 可建立二阶模型，准确描述非线性关系和交互作用
- 能够通过求解模型找到全局最优点
- 响应面图和等高线图直观、美观
- 局限性：试验次数较多（27次），当因素数>5时试验次数急剧增加

### 6.3 推荐策略

在实际工程优化中，推荐采用**序贯试验策略**：
1. **第一阶段**：使用正交设计快速筛选显著因素并确定大致最优区域；
2. **第二阶段**：在最优点附近使用响应面法（BBD或CCD）精确寻优；
3. **补充分析**：如需考察极多水平，可穿插使用均匀设计进行补充探索。

## 7 结论

本文针对"激光焊接工艺参数多方法优化与比较"这一问题，综合运用了方差分析、正交试验设计、均匀设计和响应面法等多种试验设计方法，得出以下结论：

**(1)** 通过正交试验L9(3^4)的极差分析和方差分析，确定了影响焊接接头抗拉强度的因素主次顺序为：激光功率（R=68.0）> 焊接速度（R=28.0）> 离焦量（R=22.0）> 保护气流量（R=7.0）。方差分析表明激光功率为显著影响因素（F=17.22 > F₀.₀₅(2,4)=6.94）。

**(2)** 均匀设计U9*(9^4)通过9次试验考察了每个因素的9个水平，建立的线性回归模型R²=0.9951，拟合效果良好，但模型为纯线性，无法刻画非线性效应。

**(3)** 响应面法Box-Behnken设计（27次试验）建立了包含线性项、二次项和交互项的二阶回归模型（R²=0.9931），揭示了激光功率的边际递减效应和功率-速度间的正向交互作用。通过模型求解得到全局最优解。

**(4)** 三种方法的最优参数组合高度一致：激光功率1400W、焊接速度40mm/s、离焦量+1mm、保护气流量20L/min。该一致性验证了优化结果的可靠性。

**(5)** 对三种设计方法的比较表明：正交设计适合初步筛选，均匀设计适合大范围探索，响应面法适合精确寻优。三种方法各有优势，在实际工程中应根据问题特点和研究阶段灵活选择或组合使用。
