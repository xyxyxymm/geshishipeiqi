## 四、Python脚本使用说明

### 环境要求
- Python 3.7+
- 依赖库：`numpy`, `scipy`, `matplotlib`

安装依赖：
```bash
pip install numpy scipy matplotlib
```

### 1. gen_data.py —— 数据构造与计算验证

运行此脚本可验证论文中所有计算是否正确：
```bash
python gen_data.py
```

输出内容：
- 正交设计 L9(3⁴) 数据表
- 极差分析（K值、k值、极差R）
- 方差分析表（SS、MS、F值）
- 均匀设计 U9*(9⁴) 数据表及回归方程
- 响应面 Box-Behnken 二阶模型系数

### 2. plot_figures.py —— 生成论文配图

```bash
python plot_figures.py
```

所有图片输出至 `figures/` 文件夹。如需重新生成图片，直接运行即可。

## 五、实验因素与水平

| 因素 | 符号 | 单位 | 水平1 | 水平2 | 水平3 |
|------|------|------|-------|-------|-------|
| 激光功率 | A | W | 1000 | 1200 | 1400 |
| 焊接速度 | B | mm/s | 20 | 30 | 40 |
| 离焦量 | C | mm | -1 | 0 | +1 |
| 保护气流量 | D | L/min | 10 | 15 | 20 |

响应指标：焊接接头抗拉强度 Y（MPa，望大特性）

辅助指标：焊缝宽度（mm，望目特性，目标值1.5mm）

---

## 六、主要结论

1. 激光功率是影响抗拉强度的**最显著因素**（极差R=68.0，F=17.22，p<0.05）
2. 因素主次顺序：A（功率）> B（速度）> C（离焦）> D（气流量）
3. **综合最优工艺参数**：激光功率1400W、焊接速度40mm/s、离焦量+1mm、保护气流量20L/min
4. 三种方法的最优参数高度一致，验证了结论的可靠性
5. 正交设计适合初步筛选（9次试验），均匀设计适合大范围探索，响应面法适合精确寻优（27次试验）
